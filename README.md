# 🚀 Smart Task Manager

A full-stack task management system with modern UI and powerful features.

## 🔥 Features
- 🔐 Authentication (JWT)
- 📝 Add / Edit / Delete Tasks
- ✅ Toggle Task Status
- 🔍 Smart Search (multi-word)
- 🎯 Filter (Completed / Pending)
- 🎨 Modern Glass UI (Responsive)

## 🛠 Tech Stack
- Frontend: Next.js, TypeScript
- Backend: Node.js, Express
- Database: Prisma + SQLite

## 🚀 Run Locally

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev
